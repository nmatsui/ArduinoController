Arduino Controller
==================

仮想COMポート経由でArduinoへBIT配列を送り込みます。
ただそれだけです。

Windows7 64bitで動作確認しています。

環境準備
--------
Arduinoが利用する仮想COMポートは、コントロールパネルのデバイスから調べてください。
